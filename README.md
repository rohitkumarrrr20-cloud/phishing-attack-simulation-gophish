## ğŸ” What is Phishing?

Phishing is a social engineering attack where attackers trick users into revealing sensitive information such as:
- Usernames & Passwords
- Bank Details
- Credit Card Information
- OTPs
- Personal Data

Attackers impersonate trusted organizations like banks, Google, colleges, or companies.
